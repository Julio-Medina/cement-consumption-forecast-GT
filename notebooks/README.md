# Notebooks

Suggested notebook sequence:

1. `01_data_source_review.ipynb` - inspect downloaded Excel files and identify useful columns.
2. `02_cleaning_and_target_definition.ipynb` - build the first monthly target/proxy table.
3. `03_exploratory_analysis.ipynb` - analyze seasonality, correlations, trends, and missingness.
4. `04_baseline_forecasts.ipynb` - compare naive, seasonal naive, moving average, and SARIMA.
5. `05_machine_learning_forecasts.ipynb` - add tree-based models with lag/rolling features.
6. `06_interpretability_and_report.ipynb` - feature importance, forecast interpretation, and final figures.
